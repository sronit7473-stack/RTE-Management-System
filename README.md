# RTE Management System

## 📌 Description
A Java web application using Servlets, JSP, and MySQL to manage student data with authentication and role-based access.

## 🚀 Features
- Login/Register with BCrypt
- Admin & Student roles
- Student CRUD (Add, Edit, Delete)
- Session & Filter security
- Bootstrap UI

## 🛠 Tech Stack
- Java (Servlets, JSP)
- MySQL
- Bootstrap
